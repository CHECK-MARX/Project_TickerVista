## セットアップ
1. `.env` に API キーを設定
2. `docker compose up -d`
3. `npm run dev` でフロントを起動

## 画面の歩き方
- **ダッシュボード**: 銘柄サジェストからチャート/指標を一括表示
- **マーケット概況**: インデックス・為替・ヒートマップで地合いを確認
- **学習ハブ**: Markdown で追加できるチートシート

## 小ワザ
- `⌘K / Ctrl+K` でページ・銘柄を瞬時にジャンプ
- `/symbols` で最近見た銘柄を呼び出し
- `/indicators` で指標の式と注意点をすぐ確認
