## ローソク足
- 実体の色で終値が始値より高いかを判別
- ヒゲは市場の拒否反応として解釈

## 代表的な足型
- ピンバー: 反転の兆し、必ず出来高で裏付け
- インサイドバー: ブレイク前の圧縮

## 移動平均
| 種類 | 特徴 |
| ---- | ---- |
| SMA | ノイズに強い、遅行 |
| EMA | 直近を重視、早いがダマシ多い |

## チェックボックス
- 複数時間軸でトレンド方向を合わせる
- 価格×出来高で意義を評価

> さらに詳しくは `/indicators` セクションで式と注意点を参照してください。
