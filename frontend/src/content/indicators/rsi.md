## 概要
RSIは直近の平均上昇幅と平均下落幅の比率から 0~100 のスコアを算出し、買われ過ぎ/売られ過ぎを素早く把握できます。

## 計算式
```
RSI = 100 - (100 / (1 + RS))
RS = Average Gain / Average Loss
```

## 使い方
- 70以上は過熱、30以下は売られ過ぎの目安
- トレンドラインと組み合わせてダイバージェンスを確認
- 期間を短くするとノイズが増えるため、14期間を基準に微調整します

## 注意点
> 強いトレンドでは70/30を突破したまま張り付くケースがあるため、移動平均や出来高で裏付けること。
